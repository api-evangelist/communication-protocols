# Communication Protocols

Communication protocols are systems of rules that allow two or more entities in a networked system to exchange information. Modern API platforms rely on layered protocol stacks that span the physical and link layers, network and transport layers, and application protocols such as HTTP, gRPC, MQTT, AMQP, WebSockets, SSE, and Webhooks.

## APIs

### Hypertext Transfer Protocol (HTTP)
The application layer protocol that powers the World Wide Web and the majority of REST and HTTP APIs. HTTP/1.1, HTTP/2, and HTTP/3 over QUIC are defined by IETF.

**Human URL:** [https://httpwg.org/specs/](https://httpwg.org/specs/)

#### Tags
- Application Protocol, HTTP, IETF, REST, Web

### gRPC
A high-performance, contract-first RPC framework built on HTTP/2 with Protocol Buffers as the default IDL.

**Human URL:** [https://grpc.io/](https://grpc.io/)

#### Tags
- CNCF, HTTP/2, Protocol Buffers, RPC, Streaming

### GraphQL
A query language and runtime for client-driven data APIs over HTTP and WebSockets.

**Human URL:** [https://graphql.org/](https://graphql.org/)

#### Tags

### WebSocket
A bidirectional, full-duplex communication protocol over a single TCP connection, defined in RFC 6455.

**Human URL:** [https://www.rfc-editor.org/rfc/rfc6455](https://www.rfc-editor.org/rfc/rfc6455)

#### Tags
- Bidirectional, IETF, Real-Time, Web, WebSocket

### MQTT
A lightweight publish-subscribe messaging protocol designed for constrained devices and unreliable networks. OASIS standard.

**Human URL:** [https://mqtt.org/](https://mqtt.org/)

#### Tags
- IoT, Lightweight, OASIS, Pub/Sub

### AMQP
The Advanced Message Queuing Protocol, an OASIS standard for interoperable, broker-mediated messaging.

**Human URL:** [https://www.amqp.org/](https://www.amqp.org/)

#### Tags
- Brokered, Enterprise Messaging, OASIS, Queuing

### Webhooks
An HTTP-based pattern for delivering event notifications from one service to another by performing an outbound HTTP POST.

**Human URL:** [https://webhooks.fyi/](https://webhooks.fyi/)

#### Tags
- Async, Events, HTTP, Push

### Server-Sent Events (SSE)
A simple, one-way, server-to-client streaming protocol defined as part of the HTML living standard.

**Human URL:** [Server-sent events specification](https://html.spec.whatwg.org/multipage/server-sent-events.html)

#### Tags
- HTML, HTTP, Real-Time, Streaming
